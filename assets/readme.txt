This folder contains some static assets for the application.

create.sql contains the SQL code to create the SQLite database structure. This is necessary to create the database used by the application.

sources.json is a dump of Joomla! sources up to an including Joomla! versions 5.0.0 and 4.4.0 released in October 2023. This is necessary because older Joomla! versions (released between 2005 and 2012) are not tagged and/or not available for download from Joomla's GitHub repository. The download sources for these old versions were collected manually from https://downloads.joomla.org.
